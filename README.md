A full-stack, cross-platform gifting and e-commerce mobile app that lets users discover, customize, and purchase gifts for corporate clients, friends, and family. Built with **React Native (Expo)** on the frontend and **Express.js** on the backend, integrating **Firebase Authentication**, **MongoDB**, and the **Razorpay Payment Gateway** for secure logins, seamless payments, and order management.

Key features: authentication, category-based product browsing, personalized gifting, shopping cart, address management, order tracking, and secure checkout. Backend deployed on **Render**; app distributed via **Expo** during development, with the release build prepared for the **Google Play Store**.
